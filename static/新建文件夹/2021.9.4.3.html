使用 os.path.splitext(file)[0] 可获得 文件名 。 使用 os.path.splitext(file)[-1] 可获得以 . 开头的 文件后缀名 。

code

​x
import os
file = "Hello.py"
​
# 获取前缀（文件名称）
assert os.path.splitext(file)[0] == "Hello"
​
# 获取后缀（文件类型）
assert os.path.splitext(file)[-1] == ".py"
assert os.path.splitext(file)[-1][1:] == "py"
